## Mini-Doc Api

### Link: https://apinodemongodb.onrender.com/

### Ferramentas: Node, Express, MongoDB, Mongoose, TypeScript, ESLint, Prettier, TSX.


### Comandos:

### Instalar dependências:
  ##### ```npm install```

### Iniciar servidor
  ##### ```npm run dev```

### Buildar aplicação
  ##### ```npm run build```

### Start aplicação de produção
  ##### ```npm start```
